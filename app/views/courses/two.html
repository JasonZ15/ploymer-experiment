Syllabus

Credit Units 3

Course Description

In this course, you will learn how to create and refine interaction designs that ensure a quality user interface. You will learn the theory behind good user interface design and develop the skills needed to design, implement, and evaluate your own user interface.


In this course, we emphasize the agile and user-centered design process and cover the complete design process cycle.

  • Requirement gathering: You will learn the importance of the user and task analysis and the techniques to perform the analysis
  • Design: Usability has several dimensions. Learnability, efficiency, and safety are the three dimensions that we highlight in this course. You will learn the design principles to make the user interface easy to learn, efficient to use, and less error-prone.
  • Prototyping: The design ideas or different design alternatives need to be quickly presented in front of the target users for validation. You will learn the techniques for rapidly prototyping user interfaces, including paper prototyping, computer prototyping, and web-based framework with model-view-controller software architectural pattern.
  • Evaluation: Evaluation is the integral part of the user-centered design process. You will learn the techniques for evaluating and measuring the interface usability, including heuristic evaluation and formative evaluation.

Prerequisites: None

Faculty Availability

Course Learning Outcomes

  1. Name usability dimensions and explain each dimension in details
  2. Describe the iterative user-centered design process
  3. Perform user and task analysis
  4. Know UI design principles and guidelines and design usable interfaces using them
  5. Implement UI prototype
  6. Evaluate usability of UI prototype and product
